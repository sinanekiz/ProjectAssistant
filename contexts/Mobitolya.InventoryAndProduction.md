# Mobitolya Inventory And Production

## Main Domain Areas

Observed operational areas include:

- warehouses
- warehouse stock
- production
- production statistics
- company/branch/location data
- address and operational organization

## Likely Question Topics

- stock mismatch
- warehouse movement confusion
- production step/state issue
- branch/company configuration issue
- inventory visibility problem

## Practical Rule

If the question is about what exists physically or operationally in stock,
warehouse, or production flow, prefer this context before treating it as a pure
finance or order problem.
