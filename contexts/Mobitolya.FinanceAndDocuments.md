# Mobitolya Finance And Documents

## Main Domain Areas

Observed finance/document areas include:

- invoices
- incoming invoices
- financial accounts
- financial items
- financial plans
- financial transactions
- financial documents
- payment requests
- cash accounts and transactions

## Likely Question Topics

- invoice not created or not progressing
- payment request confusion
- cash/account movement mismatch
- financial document due/reminder behavior
- accounting flow questions

## Response Guidance

These questions often need entity-aware answers.

Prefer answering in terms of:

- which document/entity is involved
- which status/step likely failed
- what user should verify next
