# Mobitolya Integrations And Messaging

## Main Domain Areas

Observed integration/messaging areas include:

- WooCommerce
- Shopify
- mail/SMS/messaging
- AI assistant endpoints
- AI OData query assistance
- messaging workers
- incoming order parsing workers

## Likely Question Topics

- integration sync not working
- imported order/product mismatch
- AI assistant answer issue
- WhatsApp-like incoming support/request parsing issue
- notification or message delivery issue

## WhatsApp Interpretation Rule

Because a large part of support arrives through WhatsApp, many support messages
may describe symptoms rather than precise technical causes.

The assistant should first normalize the message into one of these buckets:

- usage help
- parsing misunderstanding
- workflow bug
- integration sync issue
- feature request
