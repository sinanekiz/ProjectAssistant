# Mobitolya Orders And Requests

## Main Domain Areas

Observed order/request areas include:

- customer orders
- incoming orders
- offers
- product/category management
- sales channel interactions
- AI-assisted incoming request parsing

## Likely Question Topics

- incoming order did not parse correctly
- customer request from WhatsApp was misunderstood
- order status or transition issue
- offer/order conversion behavior
- product/category mismatch in order flow

## Important Interpretation Rule

If a customer message is about "what the customer wanted" versus "what the
system created", the likely focus is incoming order parsing or order workflow.
